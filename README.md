# DiscourseLens Android

DiscourseLens is a Jetpack Compose Android app for AI-powered writing and discourse analysis.

## Features

- Text input analysis flow
- File upload screen
- Animated loading screen
- Results dashboard
- Paragraph-level analysis
- Detailed feedback screen
- Light and dark mode
- Animated sun/moon theme toggle

## Tech Stack

- Kotlin
- Jetpack Compose
- Navigation Compose
- Material 3

## Getting Started

1. Open the project in Android Studio
2. Sync Gradle
3. Run on emulator or device
